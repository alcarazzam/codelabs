Muchas veces queremos obtener los datos de un solo conjunto, por ejemplo, los clientes que viven en Nueva York. Para ello se utiliza el comando WHERE. Este se coloca después del FROM y va seguido de la condición que queremos que se cumpla. Si son más de una, se usa el operador AND. Un ejemplo es el siguiente:

SELECT 
    *
FROM
    customers
WHERE
    city = 'New York'

En la consulta de arriba hemos solicitado todas las columnas de la tabla customers donde su ciudad sea Nueva York. Dicho de otra forma, la información de los clientes de Nueva York.

Si te fijas, puedes deducir la sintaxis de las condiciones. Estas siguen el patrón nombre_de_la_columna operador valor. Aquí hemos utilizado el operador de igualdad (=). La lista completa de operadores es la siguiente:

Es posible usar múltiples condiciones con los operadores AND y OR. Veamos cada uno por separado.

AND

AND sirve para indicar que dos condiciones deben cumplirse al mismo tiempo para que sea válido. Un ejemplo es el siguiente:

SELECT 
    *
FROM
    orders
WHERE
    shipping_fee > 100
        AND payment_type = 'Credit Card'

OR

OR sirve para indicar que con que una de las condiciones se cumpla el valor es válido. Un ejemplo es el siguiente:

SELECT 
    *
FROM
    orders
WHERE
    ship_city = 'Las Vegas'
        OR ship_city = 'New York'
